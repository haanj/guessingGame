How it works

The JavaScript on this page is really simple atm. The script prompts the user for their name and stores it into a variable called userName. The next piece of script alerts the user with a custom message that concatenates userName to the message for a personalized effect.

I put all the questions and answers into an array, designing the script so that I could add/remove any number of Q/A sets without breaking the program.

A for loop brings the user through each question in questionArray. The user is prompted for each question, and this is stored into a variable, then run through a basic error correction function before being checked against the correct answer.

If the user is correct, correctCount increments, and the user is alerted that they were correct. If the user was incorrect, the user is just alerted that they were incorrect.

After the script runs through the questionArray, the user is alerted their score, and an if/else statement branch alerts another message depending on their score.

The bonus round stores a random number between 1 and 10 into randomNumber. Then the user is prompted to guess, and that guess is stored into the new variable numberGuess. numberGuess is run through parseInt(), which converts it from a string

While the user's answer does not equal the random number, the user will be prompted for another guess and told whether their previous guess was too high or too low. When their answer is correct, the loop will break and they'll be congratulated
