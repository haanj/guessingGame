guessingGame
-------------
author: Joshua Haan


-------------
Description
-------------

This project was designed as part of a Code Fellows assignment. When loaded in a browser, the user will be asked a series of pre-determined questions. First, the user is asked a series of yes/no questions, then scored off their result. Then for the second part, the user is asked to guess numbers.
